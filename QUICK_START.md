## Using Rich Text Blocks

### Creating a Rich Text Block

1. Open any board in edit mode
2. Click the **"Add Block"** button at the bottom
3. In the Content section, select **"Rich Text"**
4. Start typing in the editor

## Editing Existing Blocks

### All Block Types

Every block now has three controls that appear on hover:

1. **Edit (‚úèÔ∏è)**: Click to edit the block inline
2. **Visibility (üëÅÔ∏è)**: Toggle to show/hide on public view
3. **Delete (üóëÔ∏è)**: Remove the block (with confirmation)

### Block-Specific Editing

#### Rich Text Block

- Click the Edit button or click on the text
- Editor opens inline with full toolbar
- Click "Done" when finished

#### Link Block

- Edit title, URL, description
- Add emoji icons (just paste the emoji!)
- Changes save automatically

#### Button Block

- Customize text and URL
- Choose from 4 styles: Primary, Secondary, Outline, Ghost
- Select size: Small, Medium, Large

#### Image Block

- Update image URL and alt text
- Add optional caption
- Set aspect ratio: Auto, Square, Portrait, Landscape
- Add optional link URL

## Reordering Blocks

### Drag & Drop

1. Hover over any block in edit mode
2. Grab the **‚ãÆ‚ãÆ** handle on the left
3. Drag up or down to reorder
4. Release to drop in new position

### Keyboard Navigation

- Use Tab to navigate between blocks
- Use arrow keys while holding Space to reorder

## Editing Board Title & Description

1. Click on the board title or description
2. Edit inline
3. Click outside or press Tab to finish
4. Changes save when you click "Save"

## Tips & Best Practices

### Rich Text Tips

- Use headings to create visual hierarchy
- Keep paragraphs short for readability
- Use lists to break down information
- Add links to external resources
- Use blockquotes for testimonials or emphasis

### Organization

- Use dividers to separate sections
- Add spacers for breathing room
- Hide blocks you're working on with the visibility toggle
- Reorder blocks to perfect your layout

### Performance

- Rich text blocks are optimized for performance
- Images lazy load automatically
- Save frequently to avoid losing work

## Keyboard Shortcuts (in Rich Text Editor)

- **Bold**: Cmd/Ctrl + B
- **Italic**: Cmd/Ctrl + I
- **Underline**: Cmd/Ctrl + U
- **Undo**: Cmd/Ctrl + Z
- **Redo**: Cmd/Ctrl + Shift + Z
- **Create Link**: Cmd/Ctrl + K

## Troubleshooting

### Editor Not Loading?

- Refresh the page
- Check your internet connection
- Clear browser cache

### Can't Save Changes?

- Make sure you're logged in
- Check you have edit permissions
- Click the Save button in the top toolbar

### Blocks Not Reordering?

- Make sure you're in edit mode
- Try refreshing the page
- Grab the handle on the left side
